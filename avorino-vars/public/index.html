Custom Code Instructions (after building page)

The Designer Extension API cannot set page custom code. You must add these manually:

1. Page Settings → Custom Code → Head Code 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Rejhinald/avorino@COMMIT/avorino-about-head.css">
2. Page Settings → Custom Code → Footer Code 
<script src="https://unpkg.com/lenis@1.1.18/dist/lenis.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/ScrollTrigger.min.js"></script>
<script src="https://cdn.jsdelivr.net/gh/Rejhinald/avorino@COMMIT/avorino-about-footer.js"></script>
3. Set breakpoints manually in Designer

Select each section/grid, switch to Tablet/Mobile breakpoints, and adjust padding & grid columns.
